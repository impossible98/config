# Vim

## Installation

``` bash
# 安装必要的一些系统工具
sudo apt-get update
sudo apt-get install --no-install-recommends \
    --yes \
    vim
# 下载配置
# Github
curl -# -o ~/.vimrc https://raw.githubusercontent.com/impossible98/config/master/src/vim/.vimrc 
# Gitee
curl -# -o ~/.vimrc https://gitee.com/impossible98/config/raw/master/src/vim/.vimrc
```
